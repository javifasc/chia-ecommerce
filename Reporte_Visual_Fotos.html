info 1. Explicación General sobre las Fotos

¿Por qué hay productos cargados en la web que no tienen foto?

Los productos de la tienda se importaron masivamente a la base de datos a partir de la planilla de inventario del negocio (Excel). Para que cada producto muestre su foto, debe existir un archivo de imagen en la carpeta local que coincida exactamente con su código único (ej: ACE01.jpg para el producto con código ACE01).

¿Cómo funciona el logotipo provisional?

Para aquellos productos que no tienen su foto correspondiente en la carpeta local, el sistema muestra automáticamente el **logotipo oficial de #CHIA**. Esto asegura una experiencia de usuario sumamente profesional, prolija y limpia, evitando enlaces rotos, cuadros vacíos o imágenes de error genéricas.
